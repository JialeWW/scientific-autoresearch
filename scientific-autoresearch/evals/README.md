# Historical Benchmark Inputs

The JSON files in this directory are byte-frozen case sources for benchmark protocol 2.1.2 and Skill 0.2.8. Their paths and hashes are part of that immutable historical protocol line.

They are not v0.3.1 runtime instructions, current examples, or a behavioral result. Agents applying v0.3.1 should follow `SKILL.md` and its routed references, not infer current workflow semantics from these legacy cases.

Current v0.3.x development cases live at repository level under `benchmarks/development-cases/`. They remain unscored and unevaluated. A future protocol may relocate its own copied case source, but must not rewrite the frozen 2.1.2 artifacts.
