# Claim Types

Use this file to match the scientific claim to an appropriate test.

## Detection

Question: does a feature, signal, object, event, or component exist?

Evidence should include:

- Feature or signal model.
- Background/noise model.
- Detection threshold.
- Artifact checks.
- Sensitivity or upper limit if nondetected.

## Measurement

Question: what is the value and uncertainty of a quantity?

Evidence should include:

- Measurement model.
- Error propagation.
- Calibration assumptions.
- Sensitivity to method choices.

## Association

Question: does one quantity vary with another?

Evidence should include:

- Mechanistic sign expectation.
- Supported sample.
- Confounder checks.
- Appropriate statistic.
- Per-case diagnostics.

## Causal or Mechanistic Inference

Question: does a mechanism explain the observation?

Evidence should include:

- Competing mechanisms.
- Necessary versus sufficient distinction.
- Controls that target alternative explanations.
- Model predictions beyond the fitted data.

## Model Comparison

Question: which model better explains or predicts the data?

Evidence should include:

- Same data for all models.
- Comparable likelihood or predictive metric.
- Penalty for flexibility.
- Residual or posterior predictive checks.

## Population Inference

Question: what distribution, rate, fraction, or scaling relation describes the population?

Evidence should include:

- Selection function.
- Censoring or nondetection treatment.
- Measurement errors.
- Intrinsic scatter or population variance.

## Method Validation

Question: does the analysis method work?

Evidence should include:

- Simulated or known-truth cases.
- Injection/recovery where applicable.
- Failure examples.
- Sensitivity to assumptions.

## Anomaly Discovery

Question: which cases violate the current model?

Evidence should include:

- Baseline model.
- Outlier definition.
- Data-quality checks.
- Plausible physical and nonphysical explanations.
