# Literature and Evidence Premise

Use this file when a mechanism depends on prior work, a result may be promoted as novel, or literature conflict affects interpretation.

## 1. Define the Search Scope

Record:

- scientific question and concepts;
- databases, indexes, repositories, or other sources searched;
- exact queries and filters;
- search dates and coverage limits;
- inclusion and exclusion rules;
- whether the search is targeted, scoping, systematic, or another declared type.

Do not call a search systematic unless the method meets an appropriate systematic-review standard.

## 2. Verify Sources

- Prefer primary evidence for scientific claims and authoritative records for standards, datasets, methods, and corrections.
- Verify title, authors, persistent identifier, version, publication status, and retraction or correction status.
- Distinguish preprints, peer-reviewed articles, reviews, conference material, datasets, code, and commentary.
- Do not cite a source that was not inspected sufficiently to support the claim.
- Do not infer novelty from absence in a narrow or failed search.

## 3. Build an Evidence Premise

For every premise used to motivate or interpret the current claim, record:

```text
premise_id
premise
source_id
evidence_type
population_or_system
method
direction_and_scale
main_limitations
supports_conflicts_or_null
verification_status
```

Include conflicting, null, and methodologically limiting evidence rather than only supportive results. Separate established, disputed, indirect, and unknown premises.

## 4. Connect Literature to the Autoresearch Loop

- Use prior work to define plausible mechanisms, meaningful effect scales, controls, and failure modes.
- Do not use literature after the fact only to rationalize a discovered pattern.
- Freeze literature-derived thresholds, windows, or priors before confirmation when possible.
- If a result conflicts with prior work, test whether population, support, measurement, scale, or model differences explain the conflict before claiming novelty.
- Before promotion, state exactly what is new: data, measurement, method, population, scale, mechanism test, or independent replication.

## 5. Preserve Reproducible Literature Artifacts

For literature-dependent runs, save or reference:

- `search_protocol.md`;
- `literature_inventory.csv` or equivalent;
- a citation file in a standard format when needed;
- notes linking each major claim to its supporting source;
- unresolved conflicts, unavailable sources, and search limitations.

Do not fabricate citations, quotations, identifiers, access dates, or conclusions.
